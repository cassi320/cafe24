# 자사몰 프로모션 대시보드

> **파일:** `프로모션_대시보드.html` (단일 파일, 별도 설치 불필요)
> **데이터 저장:** 브라우저 `localStorage` — 새로고침해도 유지됨
> **디자인:** Toss UI 스타일

---

## 탭 구성

### 탭 1 — 대시보드

#### 월별 캘린더
- 프로모션 기간을 색상 막대로 표시 (기간 전체에 걸쳐 표시)
- 오늘 날짜 하이라이트 (파란 원)
- 여러 프로모션이 겹치는 날은 막대를 세로로 쌓아 표시
- 이전달/다음달 네비게이션
- 막대 클릭 시 해당 프로모션 카드로 스크롤

#### 프로모션 카드 (2열 그리드)
각 프로모션마다 카드 1개 생성되며, 아래 내용 포함:

**혜택 정리**
| 항목 | 설명 |
|------|------|
| 행사 상품 | 정가 / 할인가 / 할인율(정가 대비, 소수점 반올림) |
| 무료배송 | 상품별 개별 설정, 순이익에서 −3,000원 반영 |
| 증정품 | 회원한정 / 전고객 구분 표시, 수량 × 원가 반영 |
| 순이익 | 판매가 − 제품원가 − 배송비(무료 시) − 증정품원가(수량 반영) |
| 회원 전용 혜택 | 회원가입 쿠폰팩 등 별도 박스로 표시 |

> **증정품 원가 반영 방식**
> - 증정품은 "적용 상품"을 개별 설정
> - 적용된 상품 각각의 순이익에서 증정품 전체 원가(수량 × 단가)를 차감
> - 균등 분배 아님 — 상품별 독립 차감

**준비사항 체크리스트**
- 기본 항목 (프로모션 생성 시 자동 추가):
  - 메인배너, 상세배너, 구매하기 말풍선, 혜택썸네일
  - 디자인 요청, 퍼블리싱 요청
  - 시작일 카카오톡 플친메시지
  - 종료일 데이터라이즈 메시지
  - 프로모션 소재
- 완료 체크 시 취소선 처리, 진행률 바 자동 업데이트
- 항목 직접 추가 (입력 후 Enter 또는 `+` 버튼)
- 항목 삭제 버튼 (hover 시 노출)
- **localStorage 자동 저장**

### 탭 2 — 프로모션 추가/편집

#### 입력 항목
| 항목 | 설명 |
|------|------|
| 프로모션명 | 카드 제목 |
| 기간 | 시작일/종료일 + 시작시간/종료시간 (00시 미만 설정 가능) |
| 색상 | 프리셋 6종 + 커스텀 컬러피커 |
| 혜택 태그 | `#증정` `#할인` `#1+1` `#1일특가` `#회원혜택` 등 다중 선택 |
| 카카오톡 플친 발송일 | 캘린더에 반영 |
| 데이터라이즈 발송일 | 캘린더에 반영 |
| 회원 전용 혜택 | 텍스트 자유 입력 |

> **시작일 변경 시 자동 연동**
> 시작일을 바꾸면 종료일 / 플친 발송일 / 데이터라이즈 발송일이 동일한 간격(일수 차)으로 자동 이동

#### 행사 상품 선택
- 상품 목록에서 체크박스로 선택/해제
- 프로모션 판매가 입력 → 할인율 자동 계산
- 최저판매가 미만 설정 시 경고 표시

#### 증정품 설정
- 증정품 추가 버튼으로 행 단위 추가
- 각 증정품 행에서:
  - 증정품 선택 (상품 관리 탭에서 관리)
  - 수량 입력
  - 혜택 유형 선택: `전고객` / `회원한정`
  - **적용 상품 다중 선택** (체크박스, 행사 상품 중 선택)
- 증정품 삭제 가능

#### 체크리스트 구성
- 기본 항목에서 포함/제외 선택 가능
- 저장 후 탭 1 카드에서 항목 추가/삭제 가능

#### 추가 기능
- **편집:** 기존 카드의 편집 버튼으로 진입
- **복제:** 기존 카드를 복사해 새 프로모션 초안 생성 (독립된 데이터)
- **삭제:** 카드 삭제 (되돌리기 없음)

### 탭 3 — 상품 및 증정품 가격 관리

#### 상품 목록
| 컬럼 | 설명 |
|------|------|
| 썸네일 | 클릭해서 이미지 등록 가능 |
| 상품명 / 카테고리 / 용량 | 기본 정보 |
| 정가 | 공식 판매 정가 |
| 상시판매가 | 평소 자사몰 판매가 |
| 최저판매가 | 할인 한도선 (프로모션에서 이 이하로 설정 시 경고) |
| 원가 | 제품 원가 |
| 마진율 | `(상시판매가 - 원가) / 상시판매가 × 100` 자동 계산 |

- 마진율 색상 구분: 🟢 높음 / 🟡 보통 / 🔴 낮음
- 행 클릭 시 인라인 편집 모드
- 상품 추가 / 삭제

#### 증정품 목록
- 증정품명 / 용량 / 단가(원가) 관리
- 탭 2 증정품 선택 시 이 목록에서 불러옴

> **모든 변경사항 localStorage 자동 저장**

---

### 탭 4 — 프로모션 기획

#### 운영 원칙 (상단 고정)
- 상시판매가가 기본 판매가
- 1일특가 등 스팟 프로모션 제외 시 최저판매가 이상 유지
- 단기간 내 유사 혜택 중복 지양

#### 인사이트 메모
- 각 프로모션별 결과/메모 텍스트 직접 입력
- 저장 버튼으로 localStorage 저장
- 다음 기획 시 참고용

#### 프로모션 타임라인
- 현재 등록된 프로모션 기간을 3개월 타임라인으로 시각화
- 겹치는 구간 한눈에 파악

#### 기획 초안 제안
- 현재 프로모션 현황과 운영 원칙을 기반으로 다음 프로모션 초안 제시
- 각 제안 카드에는: 기획명 / 추천 기간 / 혜택 유형 / 추천 상품 / 기획 의도 포함
- "이 기획으로 추가" 버튼 → 탭 2로 이동하며 초안 자동 입력

## 실행 방법

브라우저에서 `프로모션_대시보드.html` 파일 직접 열기 (서버 불필요)

```
파일 탐색기 → 프로모션_대시보드.html 더블클릭
```

또는 로컬 서버로 실행:
```
python -m http.server 3000
→ http://localhost:3000/프로모션_대시보드.html
```
